# Gestión del Proyecto - Jump King

## 1. Información General del Proyecto

**Nombre del Proyecto:** Jump King

**Tipo de Proyecto:** Videojuego de entretenimiento - Plataformas

**Género:** Plataformas en 2D con mecánicas de salto

**Plataforma:** Web (HTML5, CSS3, JavaScript)

**Duración del Desarrollo:** Segundo Bimestre

**Estado del Proyecto:** Completado - Fase de Implementación

## 2. Descripción del Proyecto

Jump King es un videojuego de plataformas vertical inspirado en el juego original del mismo nombre. El jugador controla un personaje que debe ascender a través de 10 niveles progresivos saltando sobre plataformas, cada una con ambientes temáticos diferentes. El objetivo principal es alcanzar la cima del castillo sin caer.

## 3. Objetivos Principales

1. Implementar un sistema de física coherente con gravedad y saltos realistas
2. Crear 10 niveles con dificultad progresiva
3. Desarrollar un sistema de colisiones preciso para interacción con plataformas
4. Integrar assets visuales de calidad (fondos, personaje)
5. Implementar interfaz de usuario intuitiva con menú y HUD
6. Incorporar retroalimentación de audio mediante efectos de sonido

## 4. Alcance

### Incluido en el Proyecto

- Motor de juego en JavaScript vanilla
- 10 niveles jugables con diseño fijo
- Menú principal y pantalla de instrucciones
- Sistema de puntuación y nivel actual
- Pantalla de Game Over con resultados
- Efectos de sonido para saltos
- Integración de assets personalizados

### Excluido del Proyecto

- Persistencia de datos (guardado de partidas)
- Multijugador
- Logros o estadísticas avanzadas
- Versión para dispositivos móviles
- Traducción a múltiples idiomas
